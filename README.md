##一. 簡介

運用React Route + Redux + Express 設計出 React Server Rendering（isomorphic）機制，以達到 SEO 優化效果

##二. 安裝

	1. git clone
	2. npm install
	3. npm run dev
	4. open http://localhost:9000

  ps. npm test 可以進行 React 單元測試，由於目前部分項目修改中，因此測試不會過
